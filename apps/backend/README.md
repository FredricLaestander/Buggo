# Elysia with Bun runtime

## Getting Started

To get started with this template, simply paste this command into your terminal:

```bash
bun create elysia ./elysia-example
```

Start Prisma postgres:

```bash
bunx prisma dev
```

Start Prisma Studio:

```bash
bunx prisma studio
```

## Development

To start the development server run:

```bash
bun run dev
```
